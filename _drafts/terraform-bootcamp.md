## Static Website

Download Css Template
```bash
wget https://www.tooplate.com/zip-templates/2137_barista_cafe.zip && unzip *.zip -d . && rm -rf *.zip
```

Install http-server

```bash
npm install http-server
```

Run Web Server

```bash
http-server -p 3000
```
